# INSTAFRAME 📸 - 拍立得氛围感图片生成器

INSTAFRAME 是一款专为拍立得爱好者设计的轻量级 Web 工具。它可以将你扫描好的实体拍立得相纸，快速转化成带有「模糊背景 + 实体相纸叠加」效果的艺术展示图，非常适合分享至社交平台。

本项目采用**单文件架构**，所有 HTML、CSS 和 JavaScript 均集成在一个文件中，无需安装，即开即用。

## 🔒 隐私与安全承诺

- **100% 本地运行**：本应用没有任何后端程序  
- **绝不上传云端**：所有图像处理（裁剪 / 模糊 / 合成 / 文字绘制）均通过浏览器 Canvas API 本地完成  
- **隐私安全保障**：您的照片不会离开设备，实现物理级隐私保护

## ✨ 核心功能 (Features)

### 🖼️ 经典画幅适配
- 内置三种官方比例：
  - Instax Mini（23:31）
  - Square（1:1）
  - Wide（99:62）
- 支持自由裁剪

### ⚡ 智能边缘检测
- 提供「自动检测」功能
- 一键识别扫描件中的相纸区域
- 减少手动调整成本

### 🎛️ 实时氛围调节
- **模糊 (Blur)**：增强背景景深
- **暗度 (Dim)**：突出主体
- **前景比例 (Foreground)**：调整相纸占比
- **拟真阴影 (Shadow)**：增强层次与真实感

### ✍️ 复古排版与字体
- **打字机文案**：`Special Elite` 字体
- **手写体日期**：`Caveat` 字体
- 支持自动生成日期或手动输入

## 🕹️ 使用指南 (How to Use)

1. **导入图片**
   - 拖拽扫描好的拍立得照片到页面

2. **框选区域**
   - 调整橙色选框，仅覆盖相纸中的图像区域（不含白边）

3. **调整效果**
   - 选择画幅比例
   - 调整模糊、亮度、阴影等参数
   - 输入自定义文字

4. **导出图片**
   - 点击「下载 PNG」
   - 获取最终高清成品
